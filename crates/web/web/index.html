Accepts a full component or a core module with a component-type custom section. Each loaded contract gets its own Web Worker; within a contract, every exported instance that owns a resource is listed: call a constructor (e.g. new) to mint a handle, then invoke its methods. Per-contract trace logs appear at the bottom of each card.
